# Tabs

## Installation

Using the react components:

    import Toast from '@vividseats/vivid-design-patterns/dist/molecules/Toast';

Using the scss files:

    @import "@vividseats/vivid-design-patterns/src/styles/toast.scss";

## Basic Usage

<Playground>
    <h4>Look in the bottom right corner. For demonstration, the component is on a 2 second cycle </h4>
    
    <StatefulComponent initialState={{ isOpen: true }}>
        {({ state, setState }) => {
            setTimeout(() => {
                setState({ isOpen: !state.isOpen });
            }, 2000);
            return (
                <Toast isOpen={state.isOpen}>
                    <p>toast content</p>
                </Toast>
            );
        }}
    </StatefulComponent>
</Playground>
