# Grid

## Installation

Using the react components:

    import {Column, Row} from '@vividseats/vivid-design-patterns/dist;

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/grid.scss';

## Responsive flex-wrap and flex-direction

If you want different flex-wrap and flex-direction properties for different break points, pass in an array to flexWrap and flexDirection properties. The items in the array correspond to mobile, tablet, and desktop breakpoints.
For the flexWrap, passing in a value will apply the value to all breakpoints.


<Playground>
    <>
        FlexDirection column-reverse on mobile and tablet only
        <Row flexDirection={["row-reverse", "row-reverse", "row"]}>
            <Column sm={4} md={4} lg={4} xl={4}>
                <GridItem>1</GridItem>
            </Column>
            <Column sm={4} md={4} lg={4} xl={4}>
                <GridItem>2</GridItem>
            </Column>
            <Column sm={4} md={4} lg={4} xl={4}>
                <GridItem>3</GridItem>
            </Column>

            <Column sm={12} lg={6}>

                <GridItem/>
            </Column>
            <Column sm={12} lg={6}>
                <GridItem/>
            </Column>
        </Row>
        No Wrap on mobile only
        <Row flexWrap={["nowrap", "wrap", "wrap"]}>
