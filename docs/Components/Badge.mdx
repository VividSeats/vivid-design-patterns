# Badge
## Installation
Using the react components:

    import Badge from '@vividseats/vivid-design-patterns/dist/atoms/Badge';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/badge.scss';

## Props
<PropsTable of={Badge}/>

## Basic Usage
<Playground>
  <Badge>Event started</Badge>
  <Badge large>Event started</Badge>
</Playground>

### Prop: type
#### one of `angled`, `count`, `rounded`
<Playground>
  <Badge type="angled">from $75</Badge>
  <Badge type="count">3</Badge>
  <Badge type="rounded">Rounded</Badge>
</Playground>

### Prop: styleAs
#### one of `success`, `warning`, `error`
##### (Note: does not apply to the `angled` or `count` badge type)
<Playground>
  <Badge styleAs="success">Event started</Badge>
  <Badge styleAs="warning">Event started</Badge>
  <Badge styleAs="error">Event started</Badge>
  <Badge styleAs="success" large>Event started</Badge>
  <Badge styleAs="warning" large>Event started</Badge>
  <Badge styleAs="error" large>Event started</Badge>
</Playground>
