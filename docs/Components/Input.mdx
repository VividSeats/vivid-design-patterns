# Input
## Installation
Using the react components:

    import Input from '@vividseats/vivid-design-patterns/dist/atoms/Input';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/input';

## Basic Usage
Inputs simply use any `htmlAttributes` that you provide as props. See the `TextField` component for more detail.

<Playground style={ { minHeight: '100px', height: '100%' } }>
    <Input type="text" placeholder="Enter some text" readOnly={false} maxLength={24} />
</Playground>

## Number Input
<Playground style={ { minHeight: '100px', height: '100%' } }>
