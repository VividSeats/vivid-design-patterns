# Image

## Installation

Using the react components:
    import Image from '@vividseats/vivid-design-patterns/dist/atoms/Image';


## Basic Usage
<Playground>
    <Image src="https://a.vsstatic.com/banner/event/concerts/george-strait.jpg" alt={"All my exes live in texas"}/>
</Playground>

## Fallback Image
To provide a fallback image if the provided src fails to load, pass in the fallback prop
<Playground>
    <Image src="//a.vsstatic.com/banner/event/concerts/billie-eilish.jpgssss" fallback="https://a.vsstatic.com/banner/event/category/rock.jpg" alt="Image with fallback"/>
</Playground>

## Lazy load image via CSS
If you want to lazy load your images via css, add the loadImageViaCss prop. A width and height need to be provided either through props or through a stylesheet.

<Playground>
        <Image height="200px" loadImageViaCss src="//a.vsstatic.com/banner/event/concerts/lana-del-rey.jpg" fallback={"//a.vsstatic.com/banner/generic.jpg"} alt={"Image with fallback"}/>
</Playground>

## Lazy load image via CSS with fallback
A fallback can be provided here as well

<Playground>
      <Image height="200px" loadImageViaCss src="//a.vsstatic.com/banner/event/failure/lana-del-rey.jpg" fallback={"https://a.vsstatic.com/banner/event/category/pop.jpg"} alt={"Image with fallback"}/>
</Playground>
