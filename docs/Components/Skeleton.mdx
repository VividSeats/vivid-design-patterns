# Skeleton Loader
## Installation
Using the react components:

    import SkeletonLoader from '@vividseats/vivid-design-patterns/dist/molecules/SkeletonLoader';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/eventRows.scss';

## Props
<PropsTable of={SkeletonLoader}/>

## Basic Usage
    
<Playground>
    <SkeletonLoader isLoading={true} rowCount={5} />
</Playground>

When `isLoading` is false, the SkeletonLoader will just render its children

<Playground>
    <StatefulComponent initialState={{isLoading: true}}>
        {
            ({state, setState}) => {
                onToggle = (toggleValue) => {
                    setState({
                        isLoading: toggleValue
                    })
                }
                return (
                    <React.Fragment>
                        <SkeletonLoader isLoading={state.isLoading} rowCount={3}>
                            <EventRow subtitle="Rose Bowl Stadium - Pasadena, CA" href="#!" title="BTS - Bangtan Boys"/>
                            <EventRow subtitle="Solider Field - Chicago, IL" href="#!" title="BTS - Bangtan Boys"/>
                            <EventRow subtitle="MetLife Stadium - East Rutherford, NJ" href="#!" title="BTS - Bangtan Boys"/>
                        </SkeletonLoader>
                        <Toggle title="Toggle isLoading prop" on={state.isLoading} onToggle={onToggle} />
                    </React.Fragment>
                )
            }
        }
    </StatefulComponent>
</Playground>

## Custom Skeleton Row

You can tweak the number of rows the Skeleton row item has in the first and second column by modifying the `firstColumnLineCount` and `secondColumnLineCount` props
<Playground>
    <SkeletonLoader isLoading={true} rowCount={5} firstColumnLineCount={2} secondColumnLineCount={3}/>
</Playground>

If you want more control over the styling of you skeleton row, you can pass in your own "skeletonBone"

<Playground>
    <SkeletonLoader isLoading={true} rowCount={5} skeletonBone={<div>Loading... ‚è±</div>}/>
</Playground>
