# Button

## Installation

Using the react components:

    import Button from '@vividseats/vivid-design-patterns/dist/atoms/Button';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/button.scss';

## Basic Usage

`class="vdp-button"`

<Playground>
    <Button>Click</Button>
</Playground>

## Props

<Props of={Button} />

## Attributes

### Links

Buttons with a href attribute will render as `<a>` tags instead of `<button>` tags

### Sizing

`class="vdp-button--small"`
`class="vdp-button--large"`
