# Backdrop

## Installation

Using the react components:

    import Backdrop from '@vividseats/vivid-design-patterns/dist/atoms/Backdrop';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/_backdrop.scss';

<Props of={Backdrop} />

Most often used implicitly inside a Modal component.

## Standalone Usage

May be used on its own.

<Playground>
    <StatefulComponent initialState={{open:true}}>
            {
        ({state, setState}) => {
            const { open } = state;

            const handleClose = () => {
                setState({open: false})
                setTimeout(() => setState({open: true}), 2000)
            }

            return (
                <Backdrop isOpen={open} onClick={handleClose} />
            )
        }
    }
        </StatefulComponent>

</Playground>
