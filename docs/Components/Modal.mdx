# Modal

### WARNING

Due to the JS enter/exit animations for the react component, use `vdp-modal` and `data-state` for plain HTML elements and the prop API `isOpen` for React components.

## Installation

Using the react components:

    import Modal from '@vividseats/vivid-design-patterns/dist/molecules/Modal';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/_modals.scss';

## Basic Example

`React: isOpen={true} HTML: className="vdp-modal" data-state="opened"`

### Modal is rendered by a component that passes in a changing isOpen prop

<Playground style={{ minHeight: '400px', height: '100%' }}>
    <StatefulComponent initialState={{ isOpen: false }}>
        {({ state, setState }) => {
            return (
                <React.Fragment>
                    <Button onClick={() => setState({ isOpen: !state.isOpen })}>Click</Button>
                    <Modal isOpen={state.isOpen} onClose={() => setState({ isOpen: false })}>
                        <Modal.Header>Modal Header</Modal.Header>
                        <Modal.Body>
                            <BodyText importance={2}>
                                Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et
                                dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex
                                ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat
                                nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit
                                anim id est laborum.
                            </BodyText>
                        </Modal.Body>
                        <Modal.Footer>
                            <Button muted importance="text" onClick={() => setState({ isOpen: false })}>
                                Dismiss
                            </Button>
                            <Button importance="text" onClick={() => setState({ isOpen: false })}>
                                Confirm
                            </Button>
                        </Modal.Footer>
                    </Modal>
                </React.Fragment>
            );
        }}
    </StatefulComponent>
</Playground>
