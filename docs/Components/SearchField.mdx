# Search Field

## Installation

Using the react components:

    import SearchField from '@vividseats/vivid-design-patterns/dist/molecules/SearchField';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/search.scss';

### With Header Context

When nested as a child of `<Header />`, pass a `RefObject` as a prop to `<Header />` to gain access to the `input` DOM element within `SearchField`

<Playground>
        <StatefulComponent initialState={{open: true}}>
            { ({state, setState}) => {
            const ref = React.createRef();

            const handleClick = () => {
                ref.current.focus();
            }

            return (
                <>
                <Button onClick={handleClick}>Focus!</Button>
                <br />
                <br />
                    <Header inputRef={ref}>
                        <SearchField />
                    </Header>
                </>
            )
        }
    }
    </StatefulComponent>

</Playground>
