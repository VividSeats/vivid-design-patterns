# Chips

## Installation

Using the react components:

    import Chip from '@vividseats/vivid-design-patterns/dist/atoms/Chip';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/chips';

## Basic Usage

`class="vdp-chip"`

<Playground>
    <Chip label="I am a chip" />
</Playground>

## Props

<Props of={Chip} />

## Additional usages:

### With callback function

`onClose={someFunction}`

<Playground>
    <Chip
        label="I have a custom callback"
        onClose={val => {
            alert(`hello. my label is: "${val}"`);
        }}
    />
</Playground>

### With custom value

`value="different_value_than_label"`

<Playground>
    <Chip
        label="I have a custom value"
        value="different_value_than_label"
        onClose={val => {
            alert(`hello. my value is: "${val}"`);
        }}
    />
</Playground>
