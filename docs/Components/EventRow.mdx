# EventRow

## Installation

Using the react components:

    import EventRow from '@vividseats/vivid-design-patterns/dist/molecules/EventRow';

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/eventRow.scss';

## EventRow Props

<Props of={EventRow} />

# Examples

## Default

`className="vp-event-row"`

<Playground>

            title: 'Chicago Blackhawks',
            subtitle: 'United Center - Chicago, IL'
        }}
    />
</Playground>

## Passing in Venue Object

The EventRow component also takes in a venue object. If passed, it will be displayed as the subtitle on the row.

<Playground>
    <EventRow
        {...{
            href: 'javascript:alert("click")',
            title: 'Chicago Blackhawks',
            venue: {
                name: 'United Center',
                regionCode: 'IL',
                countryCode: 'US',
                city: 'Chicago'
            },
            date: new Date().toUTCString()
        }}
    />
</Playground>

## Events past current calendar year

If a event is not in the current year, the event will display with the year badge.

<Playground>
    <EventRow
        {...{
            href: 'javascript:alert("click")',
            title: 'Chicago Blackhawks',
            subtitle: 'United Center - Chicago, IL',
            date: new Date().setFullYear(new Date().getFullYear() + 1)
        }}
    />
</Playground>

## Checkbox

To allow the user to check and uncheck the event, add the `hasCheckbox` and `onChange` props
`className="vp-event-row"`

<Playground>
    <EventRow
        onChange={state => console.log(`the event row is now ${state ? '' : 'not '}checked!`)}
        hasCheckbox
        {...{
            title: 'Chicago Blackhawks',
            subtitle: 'United Center - Chicago, IL'
        }}
    />
</Playground>
