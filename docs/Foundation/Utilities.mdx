# Utilities

CSS classes to quickly apply general styles without writing new CSS.  
**Components should NOT use utilities to define styles**


## Installation

Using the scss files:

    @import '@vividseats/vivid-design-patterns/src/styles/_utilities.scss';

---

## Layout Utilities
**Structure:** {Class Base}-{Optional Responsive Appendage}  
**Example:** To align text to the right on desktop screens apply ***tar-min-d***

## Sizing Utilities
**Structure:** {Class Base}-{Size Appendage}-{Optional Responsive Appendage}  
**Example:** To add 2rem margin to the top and bottom of an element apply ***mv-2xl***

| Class Base | Property |
|:---|:---|
| **h** | Height |
| **lh** | Line-Height |
| **m** | Margin |
| **mb** | Margin-Bottom |
| **ml** | Margin-Left |
| **mr** | Margin-Right |
| **mh** | Margin Horizontal (Left and Right) |
| **mv** | Margin Vertical (Top and Bottom) |
| **p** | Padding |
| **pb** | Padding-Bottom |
| **pl** | Padding-Left |
| **pr** | Padding-Right |
| **ph** | Padding Horizontal (Left and Right) |
