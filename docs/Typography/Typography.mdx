## Installation

All Typography components can be found in the @vividseats/vivid-design-patterns/dist/atoms/ and are imported like so:

    import Component from @vividseats/vivid-design-patterns/dist/atoms/Component

The typography scss can be imported into other scss files:

    @import @vividseats/vivid-design-patterns/src/styles/typography.scss
