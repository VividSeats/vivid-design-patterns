# Link
## Installation
Using the react components:

    import Link from '@vividseats/vivid-design-patterns/dist/atoms/Link';

Using the scss files:

    @import "@vividseats/vivid-design-patterns/src/styles/link.scss";

## Basic Usage
`class="vp-type-link"`
<Playground>
  <Link href="https://www.youtube.com/watch?v=dQw4w9WgXcQ">Click me</Link>
</Playground>

<PropsTable of={Link}/>

## Add custom onClick function
`onClick=[Function]`
<Playground>
  <Link href="https://www.youtube.com/watch?v=dQw4w9WgXcQ" onClick={function(e) { e.preventDefault(); alert('Hello'); }}>Trigger alert</Link>
</Playground>

## Add custom class names
`class="vp-type-link custom"` `class="vp-type-link --inverted"` `class="vp-type-link --muted"`
<Playground>
  <Link href="https://www.youtube.com/watch?v=dQw4w9WgXcQ" className="custom">Click me</Link>
  <Link href="https://www.youtube.com/watch?v=dQw4w9WgXcQ" className="--inverted">Click me</Link>
  <Link href="https://www.youtube.com/watch?v=dQw4w9WgXcQ" className="--muted">Click me</Link>
</Playground>
