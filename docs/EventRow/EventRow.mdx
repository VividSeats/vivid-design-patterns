# EventRow
## Installation
Using the react components:

`import EventRow from @vividseats/vivid-design-patterns/dist/molecules/EventRow;`

Using the scss files:

`@import @vividseats/vivid-design-patterns/src/styles/eventRow.scss;`

# Examples

## Default
`className="vp-event-row"`

    <EventRow {...{
        href: 'javascript:alert("click")',
        title: 'Chicago Blackhawks',
        subtitle: 'United Center - Chicago, IL'
    }} />
</Playground>

## Events past current calendar year

If a event is not in the current year, the event will display with the year badge.


<Playground>
    <EventRow {...{
        href: 'javascript:alert("click")',
        title: 'Chicago Blackhawks',
        subtitle: 'United Center - Chicago, IL',
        date: new Date().setFullYear(new Date().getFullYear() + 1)
    }} />
</Playground>
