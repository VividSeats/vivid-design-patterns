## Installation
Using the react components:

    import Card from @vividseats/vivid-design-patterns/dist/molecule/Card

Using the scss files:

    @import @vividseats/vivid-design-patterns/src/styles/card.scss

### `<Card.Body/>`
`class="vp-card__body"`
<PropsTable of={Card.Body}/>
Cards that contain only native dom elements are automatically wrapped in a card `Card.Body`.

For example:

    return <Card>Hello</Card>

is the same as

    return <Card>
            <Card.Body>Hello</Card.Body>
       </Card>

## Other Card Types
### List Cards
Adds a border on the title to match borders within list items.
<Playground>
    <Card type="list">
        <Card.Header>
            <h5 className="vp-type-headline6">List Card</h5>
        </Card.Header>
        <Card.Body>
            <ul>
                <li> One item </li>
                <li> Two item </li>
                <li> Three item </li>
            </ul>
        </Card.Body>
    </Card>
</Playground>

### Anchor Cards
